## Overview

Accessing your trading tools and educational resources requires proper authentication. You use the members portal to manage your Edu Premium subscription, generate API credentials, and configure platform permissions for MT4, MT5, or TradingView.

## Logging Into the Members Area

Follow these steps to sign in and reach your dashboard.

<Steps>
  <Step title="Visit the portal" icon="external-link">
    Navigate to the secure members login page at `https://my.itradeaims.net/`.
  </Step>
  <Step title="Enter credentials" icon="key">
    Provide your registered email and password. Enable two-factor authentication for added security.
  </Step>
  <Step title="Access your tools" icon="settings">
    Once logged in, you can download indicators, view video modules, and manage API keys.
  </Step>
</Steps>

## API Keys and Tokens

You generate API credentials to connect external scripts or bots to your account.

<CodeGroup tabs="Python,JavaScript,cURL">
```python
import requests

headers = {"Authorization": "Bearer YOUR_API_TOKEN"}
response = requests.get("https://api.itradeaims.net/v1/tools", headers=headers)
print(response.json())
```
```javascript
const response = await fetch("https://api.itradeaims.net/v1/tools", {
  headers: { Authorization: "Bearer YOUR_API_TOKEN" }
});
const data = await response.json();
console.log(data);
```
```bash
curl -H "Authorization: Bearer YOUR_API_TOKEN" \
  https://api.itradeaims.net/v1/tools
```
</CodeGroup>

<ParamField header="Authorization" param-type="string" required="true">
  Include your generated bearer token in every request header.
</ParamField>

## MT4 / MT5 Setup Permissions

You must grant indicator access before installing tools such as the Banana Indicator or AIMS Algo on your trading platform.

<Tabs>
  <Tab title="MT4" icon="monitor">
    1. Open MT4 and go to File > Open Data Folder.
    2. Copy the downloaded `.ex4` file into the Indicators folder.
    3. Restart the platform and attach the indicator to your chart.
  </Tab>
  <Tab title="MT5" icon="monitor">
    1. Open MT5 and navigate to File > Open Data Folder.
    2. Place the `.ex5` file in the Indicators directory.
    3. Refresh the Navigator panel and drag the indicator onto any chart.
  </Tab>
</Tabs>
